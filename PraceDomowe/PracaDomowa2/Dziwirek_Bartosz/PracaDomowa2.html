Wstępny wykres

W poniższej pracy podjąłem się poprawy wykresu opublikowanego na stronie eurostatu jest to wykres opisujący graficznie ilość zużytych materiałów przez gospodarstwa domowe w EU.

library(jpeg)
library(devtools)
library(data.table)
library("readxl")
library("dplyr")
library("ggplot2")
library("ggthemr")
data <- read_excel("ENV_AC_MFA__custom_6658361616236149727.xlsx")

img <- readJPEG("EU-FB-DMC-2019.jpg")
value <- as.numeric(c(data[11,2], data[11,4], data[11,6], data[11,8], data[11,10],
                      as.numeric(data[11, 12]) + as.numeric(data[11,14])))
names <- c("Total","Biomass", "Metal ores", "Non-metallic minerals", "Fossil energy materials", "Other products")

data <- data.table(fuels = names, consumption = value)
percentage = round(value/value[1]*100, 2)
a = paste(str1 = percentage, str2 = "%")
ggthemr("earth", type="outer", layout="scientific", spacing=2)

x <- ggplot(data = data, mapping = aes(x = reorder(fuels, consumption), y = consumption, fill = names)) +
  geom_bar(stat = "identity") +
  scale_fill_manual(values = c("darkolivegreen4",  "darkolivegreen3", "seagreen3",
                                           'seagreen4',"springgreen2 ",  "darkgreen" ), guide = FALSE) +
  geom_text(aes(label=value), position=position_dodge(width=0.9), vjust=-0.25) + 
  geom_text(aes(label=a), position=position_dodge(width=0.9), vjust=1.25) +
  labs(title = "Domestic material consumption per person in the EU, 2019",subtitle = "Additionally showing how specific material stands among the others.",x = "materials" ,y = "consumption in tons")

Poprawki

Pierwotny wykres był kołowy, który nie sumował się do 100%, gdyż zostały pominięte dane zbyt małe, aby zostały na nim umieszczone (wyjaśnienie tego incydentu na pierwotnym wykresie zobaczyłem dopiero za trzecim razem).W związku z tym użyłem wykresu słupkowego i naniosłem odpowiednie wartości ilościowe i procentowe. Ponadto na samej obręczy wykresu stosuje się odcienie jednego koloru, którymi rozróżnia się poszczególne wykorszystane materiały, co dla osoby mającej problemy z rozróżnianiem kolorów natychmiast utrudnia jego odczytanie, a wręcz uniemożliwia, na wykresie poprawionym każdy słupek jest podpisany i w celach estetycznych obdarzony innym kolorem.
